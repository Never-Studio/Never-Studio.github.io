neverengine

Willkommen auf der Projektseite von neverengine.
Neverengine ist eine isometrische Game-Engine.

Features

  • das Erstellen pseudo-3d Objekten auf Basis von Bildern
  • Drehen und Bewegen im dreidimensionalen Raum
  • eigene isometrische Seitenverhältnisse sind möglich
  • das Layering funktioniert ohne Kacheln
  • bounding boxes (für Physics und Angeben eigener Aktionen bei Kollisonen)
  • animieren von Objekten
  • Funktionen für dreidimensionales isometrisches malen
  • Mauskoordinaten als isometrische Koordinaten erhalten (aktuell nur 2d)
  • Steuern von Objekten mit dem Keyboard ohne viel Boilerplate Code
  • Zentrieren des Canvas auf ein Objekt
  • etc. (in den Docs und Beispielen)
