Wer wir sind

Wir sind Neverstudio!

Neverstudio.de ist eine Website zur Veröffentlichung von Programmierprojekten.
Wir interessieren uns für Spieleentwicklung, KI und andere technische Themen.
Schau dir gerne unsere GitHub-Seite an oder probiere ein Projekt aus.
Du kannst uns auf Reddit und Instagram folgen.

FAQ

Wozu dient diese Website?

Neverstudio.de ist eine Webseite zur Veröffentlichung von Programmierprojekten.
Bisher haben wir zwar erst Neverword (ein Wortratespiel wie Wordle) veröffentlicht, wir planen aber noch, unsere isometrische JavaScript Engine (Neverengine) zu veröffentlichen.
Außerdem wollen wir uns noch mit KI beschäftigen (evtl. werdebn wir bald einen auf GPT-OSS basierenden Agenten veröffentlichen).
Es kann sein, dass wir irgendwann auch ein echtes Spiel veröffentlichen.

Wer hat diese Webseite erstellt?

Diese Webseite wurde erstellt und wird gewartet von zwei Brüdern, die sich für Programmierung interessieren.

Wie kann man ein Projekt ausprobieren?

Gehen sie zur Projektliste und besuchen sie die entsprechende Unterseite, indem sie auf den Projektseiten-Link klicken.
Wenn es eine Demo gibt, ist sie höchstwahrscheinlich dort zu finden. Sie können auch die Repositories auf GitHub ansehen.

Auf welchen Social Media Webseiten ist Neverstudio.de vertreten?

Wir haben Accounts auf Instagram, Reddit, Bluesky, GitHub und vielen weiteren Plattformen.
Nicht alle Accounts werden jedoch regelmäßig gepflegt.

Wie kann ich Bugs melden oder Projektvorschläge machen?

Wenn du einen Bug oder Fehler in einem Projekt findest oder einen Vorschlag für ein neues Feature oder Projekt hast, kannst du uns gerne kontaktieren.
Oder erstelle einfach eine Pull Request auf GitHub.
