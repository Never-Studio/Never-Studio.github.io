neverengine

Welcome to the neverengine project page.
The neverengine is an isometric game engine.

Features

  • creating semi-3d objects with 2d images
  • turning and moving objects in 3d space
  • custom aspect ratios
  • tile-free layering
  • bounding boxes (physics and custom collision actions)
  • object animation
  • functions for 3d isometric drawing
  • mouse coordinate to isometric coordinate conversion (currently only 2d)
  • out-of-the-box controllability of a player object via keyboard
  • centering the canvas on a specific object
  • etc. (see docs and examples)
